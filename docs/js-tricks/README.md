# JS Tricks  

## 类型转换

### 巧用 + ! & 运算符  

```js
/****  + 运算符  ****/
// 转换为number类型  
+'1' // 1
+false // 0

// 获取unix时间戳
+new Date() // 1561432286194

/****  ！运算符  ****/
// boolean类型转换
!!{} // true
!!'string' // true
!!'' // false

/****  & 运算符  ****/
// 当你想 有这个函数的时候就执行
if(fn){
  fn()
}
// 用 & 运算符
fn && fn()

```
